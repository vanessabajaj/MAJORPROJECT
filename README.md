Wanderlust - Airbnb Clone

A full-stack web application that replicates core Airbnb functionalities, allowing users to list properties, browse accommodations, and leave reviews. Built with Node.js, Express, MongoDB, and modern web technologies.

Core Functionality:
1) User Authentication - Secure signup/login with password hashing
2) Property Listings - Create, read, update, and delete property listings
3) Image Upload - Cloud-based image storage with Cloudinary
4) Review System - Star ratings (1-5) and written reviews
5) User Authorization - Users can only edit/delete their own listings
6) Responsive Design - Mobile-friendly interface using Bootstrap 5

Highlights
1) MVC Architecture - Clean separation of concerns
2) RESTful API Design - Standard HTTP methods and routes
3) Session Management - MongoDB-backed persistent sessions
4) Flash Messages - User feedback for actions
5) Input Validation - Server-side validation using Joi
6) Error Handling - Centralized error management with custom error class
7) Cascade Delete - Automatic cleanup of associated data
